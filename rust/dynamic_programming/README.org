#+title: Dynamic programming

Dynamic programming is both a mathematical optimization method and an algorithmic paradigm. The method was developed by Richard Bellman in the 1950s and has found applications in numerous fields, from aerospace engineering to economics --- Wikipedia. https://en.wikipedia.org/w/index.php?title=Dynamic_programming&oldid=1298731105
